# Thiiird Place Listening Club — Website

A static website for the Thiiird Place Listening Club (watch, listen, events, membership, Social Club, and blog). Plain HTML/CSS/JS — no build step.

## Files
- `index.html` — homepage (hero, Watch, Listen, Events, Membership, Social Club, footer)
- `blog.html` — blog index (hybrid: inline reader + dedicated post pages)
- `blog/` — individual full post pages
- `.nojekyll` — tells GitHub Pages to serve all files as-is

## Publishing updates (GitHub Desktop)
1. Open GitHub Desktop — it shows the changed files.
2. Type a short summary, then click **Commit to main**.
3. Click **Push origin**.
4. GitHub Pages rebuilds automatically in under a minute.

## Live hosting
Hosted with GitHub Pages (Settings → Pages → Deploy from branch: `main` / root).

## Notes
- Forms email submissions via Web3Forms (mailing list + Social Club application).
- The Twitch live player only renders on the hosted domain (not from a local file).
